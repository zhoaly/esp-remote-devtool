1. Project Setup

本地模式要求 Local Agent 运行在 http://127.0.0.1:8765；远端模式只使用服务器白名单工作区。
页面加载时会读取 /api/lvgl/workspaces。
只上传 UI 代码会由本地 Agent 抽取指定目录，远端自动生成浏览器模拟器外壳；完整 Web 工程要求项目本身能直接生成 HTML/WASM。
项目根目录需要包含 CMakeLists.txt。UI-only 模式只读取下方指定目录，不上传完整 PC 模拟器工程。
当前 lv_port_pc_vscode 工程没有独立 UI 目录，默认留空并调用 LVGL 内置 demo；有自定义 UI 后填相对目录。
当前默认不上传本机 SDL/HAL 代码；有自定义 UI 头文件目录后填相对路径。
